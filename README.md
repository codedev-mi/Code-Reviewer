# 🤖 AI-Powered Code Reviewer (MERN + Google Generative AI)

An **AI-driven code review web app** built with the **MERN stack** that automatically analyzes and suggests improvements for your source code.  
This project demonstrates how **artificial intelligence** can enhance developer productivity by providing intelligent code feedback and insights — powered by **Google Generative AI**.

---

## 🚀 Features

- 🔍 **AI-Based Code Review** — Uses Google Generative AI to analyze code and offer meaningful suggestions.  
- 🧠 **Smart Refactoring Hints** — Detects issues and recommends improvements in readability, performance, and structure.  
- ⚡ **Vite + React Frontend** — Fast, modern setup with responsive UI and sleek styling.  
- 💬 **AI Response Handling (POST Method)** — Secure backend-to-AI request structure for improved reliability.  
- 🎨 **Syntax Highlighting with PrismJS** — Displays beautifully formatted and colored code blocks.  
- 🧾 **Markdown Rendering** — AI feedback is rendered neatly with formatted text and styles.  
- 🔗 **Axios + CORS Integration** — Smooth client-server communication across different origins.

---

## 🏗️ Tech Stack

| Layer | Technology |
|:------|:------------|
| **Frontend** | React (Vite), PrismJS, Axios, Markdown Rendering |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB |
| **AI Integration** | Google Generative AI (Gemini API) |
| **Styling** | CSS / Tailwind (optional) |
| **Hosting (optional)** | Render / Vercel / Netlify / MongoDB Atlas |
